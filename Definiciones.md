# REDES
~~~
Son un grupo de ordenadores conectados entre si para el cambio de informacion ya sea por medio fisico o por medio de ondas electromagneticas 
~~~

# LAN
~~~ 
Local Area Network (en inglés: “Red de Área Local”). Son las redes de menor envergadura, como las que podemos instalar en nuestro departamento.
~~~
# MAN
~~~
 Metropolitan Area Network (en inglés: “Red de Área Metropolitana”). Se trata de redes de tamaño mediano, óptimas para un campus universitario o el edificio de una biblioteca o empresa de varios pisos, incluso para una porción de una ciudad.
~~~
# WAN
~~~
 Wide Area Network (en inglés: “Red de Área Amplia”). Aquí entran las redes de mayor tamaño y alcance, como las redes globales o como Internet.
~~~

# IP
~~~
Una dirección IP es una dirección única que identifica a un dispositivo en Internet o en una red local. IP significa “protocolo de Internet”, que es el conjunto de reglas que rigen el formato de los datos enviados a través de Internet o la red local.
~~~
# MASCARA 
~~~
Una máscara de red se define como una combinación de bits que funcionan para delimitar una red de ordenadores y dividir esta red en subredes. Este código de números es usado para realizar correctamente el proceso de transferencia de mensajes entre dispositivos en la misma subred. Y realiza sus labores en conjunto con el enrutador de una red.

La máscara de red también se encarga de indicarle a los diferentes dispositivos cómo está dividida la dirección IP, es decir, qué parte de esta es la correspondiente al número de la red, a la máscara de subred y al host y además, permite identificar fácilmente qué subred está siendo utilizada.
~~~
